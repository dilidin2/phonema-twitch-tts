Capabilities

Everything your stream needs to talk back.

⚡

Real-time streaming

Audio plays the moment VoxCPM2 starts generating. No waiting for full synthesis -- your chat hears responses instantly with sub-second latency.

☺

Voice cloning

Drop in any reference audio file and Phonema clones that voice. 5-30 seconds of clear speech is all you need.

⇆

Voice rotation

Cycle through multiple voices in random or sequential mode. Each redemption gets a different speaker.

⚙

Queue management

Back-pressure controlled concurrent requests. The producer/consumer pattern keeps inference at playback speed.

🔔

Auto-reconnect

OAuth tokens persist to disk. Restart the server and Phonema reconnects automatically -- no manual re-auth.

🌙

Cross-platform

Runs on CPU, NVIDIA CUDA, and AMD ROCm. The ROCm build of PyTorch even speeds up inference on AMD CPUs through MKL optimizations.

Install

Up and running in two steps.

# 1. Install PyTorch (pick your hardware)
pip install torch torchaudio --index-url https://download.pytorch.org/whl/cu124

# 2. Install Phonema dependencies
pip install voxcpm numpy fastapi uvicorn twitchAPI pyyaml \
       python-dotenv loguru sounddevice soundfile

# 3. Start the server
python main.py

# 4. Connect to Twitch (first time only)
curl -X POST http://localhost:8100/twitch/connect
